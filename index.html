GitHub Storage Setup
Audio files are hosted for free on GitHub Releases — giving you permanent public URLs that anyone can stream. Beats metadata (title, BPM etc.) lives in your browser's localStorage.
1
Go to github.com/new and create a public repo — name it anything (e.g. beats). It only needs to exist — you don't need to push any code.
2
Go to github.com/settings/tokens/new → select Classic token → check the repo scope → Generate and copy the token.
3
Paste your details below and hit Save & Test.

How it works
BPM is detected by analyzing the energy envelope of the audio and finding recurring rhythmic pulses using an autocorrelation approach. Key detection analyzes the chromagram — the distribution of pitch classes — and compares it against major and minor key profiles (Krumhansl-Schmuckler). Results are estimates and may vary with complex productions or mixed tempos.
